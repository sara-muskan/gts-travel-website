## 📖 About The Project

GTS (Global Travel Services) is a travel website concept designed to showcase destinations, travel packages, customer testimonials, and booking options through a clean and modern user interface.

The project focuses on:

- Responsive design
- Modern UI/UX
- Smooth navigation
- Mobile-friendly layout
- Interactive mobile menu
- Travel package showcase
- Destination gallery
- Customer testimonials section

---

## ✨ Features

- Sticky navigation bar
- Responsive mobile menu
- Hero section with animated route map
- Popular destinations showcase
- Travel packages section
- Customer testimonials
